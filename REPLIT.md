# Инструкция по запуску бота (Replit / Persistence Mount)

## Для платформ с persistence mount (/data)

### 1. Настройка

Отредактируйте `config.py`:
```python
BOT_TOKEN = "ваш_токен_от_BotFather"
ADMIN_IDS = [5153839634]  # Ваш Telegram ID
```

### 2. Запуск

```bash
# Установите зависимости
pip install -r requirements.txt

# Назначьте администратора (сохранится в /data)
python set_admin.py 5153839634

# Запустите бота
python main.py
```

### 3. Проверка

Бот при старте покажет:
```
Using /data for persistence mount
✅ Права на запись в /data: OK
✅ База данных: OK (пользователей: 1, администраторов: 1)
```

### 4. Миграция данных (если были JSON файлы)

```bash
python migrate.py
```

### 5. Команды

- `/start` - начать работу
- `/me` - проверить статус администратора
- `/help` - справка
- `/ping` - проверка работоспособности

## Важно!

Все данные теперь сохраняются в `/data/bot.db` — это постоянное хранилище которое:
- ✅ Сохраняется при перезапуске
- ✅ Не теряется при обновлении файлов
- ✅ Доступно для записи

## Отладка

```bash
# Проверка базы данных
python -c "from data_manager import get_user; print(get_user(5153839634))"

# Должно вывести: {'admin': 1, ...}
```
