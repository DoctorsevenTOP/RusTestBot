# Тесты на вставку букв - Кнопочный интерфейс

## Обзор

Теперь тесты на вставку букв работают полностью через **кнопки Telegram** без веб-интерфейса!

## Как пользоваться

### 🎯 Прохождение теста

1. **Нажмите кнопку** `✍️ Вставка букв` в главном меню
2. **Выберите тест** из списка inline-кнопок
3. **Заполняйте пропуски:**
   - Пропуски показаны как `【1】`, `【2】`, `【3】`
   - Заполняются последовательно (сначала 1, потом 2, и т.д.)
   - Нажимайте кнопки с буквами для вставки
   - После вставки буква появляется в тексте: `[а]`, `[н]`
4. **Нажмите** `✅ Готово` для проверки
5. **Получите результат** по каждому вопросу
6. **В конце** - общий итог с сохранением в базу данных

### 🛠️ Создание теста (только администраторы)

1. **Нажмите кнопку** `✍️ Создать вставку`
2. **Введите название теста:**
   ```
   Правописание -Н- и -НН-
   ```
3. **Введите описание** (или "пропустить"):
   ```
   Вставьте пропущенные буквы в слова
   ```
4. **Добавьте вопросы:**
   - Введите текст с пропусками `..`:
     ```
     В комнате стоял див..н, который был украше.. узором.
     ```
   - Введите правильные буквы через пробел:
     ```
     а н
     ```
   - Введите варианты для выбора (минимум 2):
     ```
     а о н е
     ```
5. **Добавьте ещё вопросы** или завершите создание

## Команды бота

### Для всех:
- `✍️ Вставка букв` (кнопка) - список тестов на вставку
- `/help` - справка по всем командам

### Для администраторов:
- `✍️ Создать вставку` (кнопка) - создать тест на вставку
- `/cancel` - отменить создание теста

## Примеры

### Пример 1: Простой вопрос

**Текст:**
```
М..чта озаряла к..мнату
```

**Правильные буквы:**
```
е о
```

**Варианты:**
```
е и о а
```

**Что увидит пользователь:**
```
📝 Вопрос 1 из 1

М..чта озаряла к..мнату

👉 Сейчас заполняется пропуск 【1】
Выберите букву:

[е] [и] [о] [а]
✅ Готово
```

После выбора "е" и "о":
```
📝 Вопрос 1 из 1

[е]чта озаряла к[о]мнату

✅ Все пропуски заполнены! Нажмите 'Готово' для проверки.
```

### Пример 2: Создание теста

**Диалог с ботом:**
```
Админ: ✍️ Создать вставку

Бот: Введите название теста

Админ: Безударные гласные

Бот: Введите описание (или 'пропустить')

Админ: Вставьте безударные гласные в корнях слов

Бот: Введите текст с пропусками (.. для пропусков)

Админ: М..чта стояла в к..мнате

Бот: Введите правильные буквы через пробел

Админ: е о

Бот: Введите варианты букв (минимум 2)

Админ: е и о а

Бот: Вопрос добавлен! Всего вопросов: 1
      Что делаем дальше?
      [➕ Добавить ещё вопрос]
      [✅ Завершить создание]

Админ: [нажимает ✅ Завершить создание]

Бот: ✅ Тест на вставку букв создан!
      ID теста: 1
      Название: Безударные гласные
      Вопросов: 1
```

## Структура данных в базе

Тест сохраняется в формате:
```json
{
  "type": "insert_letter",
  "text": "М..чта озаряла к..мнату",
  "gaps": [
    {"position": 0, "correct": "е"},
    {"position": 1, "correct": "о"}
  ],
  "letters": ["а", "е", "и", "о"],
  "description": "Вставьте пропущенные буквы"
}
```

## Как это работает

### Прохождение:
1. Бот находит все тесты с вопросами `type: insert_letter`
2. Показывает список тестов
3. При выборе - показывает вопросы по одному
4. Заменяет `..` на `【1】`, `【2】`, `【3】`
5. Пользователь заполняет по порядку
6. После каждого выбора текст обновляется
7. При нажатии "Готово" - проверка ответов
8. Показывает результат и переходит к следующему вопросу
9. В конце сохраняет общий результат

### Создание:
1. Администратор вводит название и описание
2. Добавляет вопросы через диалог
3. Бот автоматически находит пропуски `..`
4. Сохраняет правильные ответы и варианты
5. По завершении создаёт тест в базе данных

## Преимущества кнопочного интерфейса

✅ **Не нужен веб-хостинг** - всё работает внутри Telegram
✅ **Не нужно настраивать URL** - работает сразу
✅ **Проще для пользователей** - всё в привычном интерфейсе
✅ **Быстрее работает** - нет загрузки веб-страниц
✅ **Тактильная отдача** - кнопки Telegram дают обратную связь
✅ **Автоматическая адаптация** - работает на всех устройствах

## Решение проблем

### Нет тестов в списке
**Причина:** В базе нет тестов с вопросами типа `insert_letter`
**Решение:** Создайте тест через кнопку `✍️ Создать вставку`

### Пользователь не может пройти тест
**Причина:** Пользователь не зарегистрирован
**Решение:** Отправьте `/start` для регистрации

### Создание останавливается
**Причина:** Не все поля заполнены корректно
**Решение:** Следуйте инструкциям бота, используйте `/cancel` для отмены

## Расширение

### Добавить новые типы вопросов
Измените `data_manager.py` и добавьте обработку в `main.py`

### Изменить оформление
Отредактируйте тексты сообщений в `main.py`

### Добавить больше вариантов букв
Просто введите больше букв при создании вопроса
