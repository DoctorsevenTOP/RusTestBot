🚀 Insert Test App 🛠️ Test Builder App

🎯 Insert Test App

Приложение для прохождения тестов на вставку букв.

  • Список доступных тестов
  • Интерактивный интерфейс с пропусками
  • Панель с буквами для выбора
  • Мгновенная проверка ответов
  • Экран результатов с детализацией
  • Отправка результатов боту

Команда бота: /insert_test

🛠️ Test Builder App

Приложение для создания тестов на вставку букв (только для администраторов).

  • Создание названия и описания теста
  • Добавление неограниченного количества вопросов
  • Автоматическое обнаружение пропусков
  • Настройка правильных букв
  • Добавление вариантов букв
  • Предпросмотр перед отправкой
  • Отправка в бот одним кликом

Команда бота: /insert_builder

📋 Команды бота

Полный список команд для работы с Mini Apps:

/insert_test - Запустить тест на вставку букв (новый)
/insert_builder - Создать тест на вставку букв (админ)
/insert - Старый интерфейс вставки
/edit_test - Старый редактор тестов
/help - Справка по всем командам

⚙️ Настройка для Telegram

Для работы в Telegram Mini Apps необходимо:

Важно: Загрузите HTML файлы на HTTPS хостинг (GitHub Pages, Vercel, Netlify) и обновите URL в main.py
# В main.py, команда /insert_test:
app_url = "https://your-domain.com/insert_test_app.html"

# В main.py, команда /insert_builder:
app_url = "https://your-domain.com/test_builder_app.html"
Готово! После развёртывания пользователи смогут проходить тесты прямо в Telegram.
