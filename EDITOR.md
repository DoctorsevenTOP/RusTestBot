# Инструкция по развёртыванию редактора тестов

## 1. Хостинг для редактора

Разместите `editor.html` на HTTPS хостинге:

### GitHub Pages (рекомендуется)
1. Создайте репозиторий на GitHub
2. Загрузите `editor.html` и переименуйте в `index.html`
3. Включите GitHub Pages
4. URL: `https://yourusername.github.io/repo-name/`

### Vercel / Netlify
1. Загрузите файл на vercel.com или netlify.com
2. Получите HTTPS URL

## 2. Настройка URL в боте

Откройте `main.py` и найдите команду `/edit_test` (строка ~280):

```python
editor_url = "https://your-domain.com/editor.html"
```

Замените на ваш URL:
```python
editor_url = "https://yourusername.github.io/repo-name/index.html"
```

## 3. Использование редактора

### Для администраторов:

1. **Запуск:**
   ```
   /edit_test
   ```

2. **Загрузка существующего теста:**
   - Откройте вкладку "📚 Тесты"
   - Нажмите на нужный тест
   - Редактор откроется с загруженными вопросами

3. **Добавление вопросов:**
   - Переключитесь на вкладку "✏️ Редактор"
   - Нажмите "➕ Добавить вопрос"
   - Выберите тип вопроса
   - Заполните данные

4. **Вставка букв:**
   - Вкладка "✍️ Вставка букв"
   - Введите текст с `..` для пропусков
   - Укажите правильные буквы
   - Добавьте варианты
   - Нажмите "➕ Добавить в тест"

5. **Отправка в бот:**
   - Нажмите "✅ Сгенерировать JSON"
   - Нажмите "📤 Отправить в бот"
   - Бот создаст или обновит тест

## 4. Формат данных

### Одиночный вопрос:
```json
{
  "type": "single",
  "text": "Вопрос?",
  "answer": "Ответ"
}
```

### Множественный выбор:
```json
{
  "type": "multiple",
  "text": "Вопрос?",
  "options": ["В1", "В2", "В3", "В4"],
  "answer": ["В1", "В3"]
}
```

### Вставка букв:
```json
{
  "type": "insert_letter",
  "description": "Вставьте буквы",
  "text": "Текст с ___ пропусками",
  "gaps": [
    {"index": 10, "correct": "а"},
    {"index": 25, "correct": "н"}
  ],
  "letters": ["а", "о", "н", "е"]
}
```

## 5. Интеграция с ботом

Редактор отправляет данные в бот через `web_app_data`:

```python
@router.message(F.web_app_data)
async def handle_editor_result(message: Message):
    data = json.loads(message.web_app_data.data)
    
    if data.get("action") == "create_test":
        # Создание теста
        test_id = generate_test_id()
        save_test(test_id, data)
```

## 6. Обновление существующих тестов

Для обновления существующего теста:

1. Загрузите тест в редакторе
2. Добавьте/измените вопросы
3. Сгенерируйте JSON
4. Отправьте в бота

Бот определит существующий тест по ID и обновит его.

## 7. Тестирование

Для локального тестирования:

```bash
# Запустите локальный сервер
python -m http.server 8000

# Откройте в браузере
http://localhost:8000/editor.html
```

Для работы Web App в Telegram используйте ngrok:
```bash
ngrok http 8000
```

Используйте HTTPS URL от ngrok в команде `/edit_test`.
